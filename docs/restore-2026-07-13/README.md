# 삭제 과정 데이터 복원 (2026-07-13)

초창기(2026년 3월~5월 초)에는 만족도 조사가 끝난 과정을 **중복 응답 방지를 위해 삭제**했다.
2026-05-05(`842fca8`)부터 활성/종료 플래그 방식으로 바뀌어 데이터가 보존되지만,
그 이전에 삭제된 과정들은 Firestore 에서 영구 소멸된 상태였다
(PITR 비활성 · 백업 스케줄 없음 확인).

로컬에 남아 있던 당시 관리자 페이지 엑셀 export 를 근거로, 정식으로 설문을 진행했던
아래 2개 과정을 **종료된 과정(active: false)** 으로 복원했다.
테스트용 설문(강사명 a/b/c, 과정명 placeholder 등)은 복원 대상에서 제외했다.

## 복원 수준

- **예산 및 회계(1기)**: 응답자별 객관식(Q1~Q9)·인구통계(Q11~Q16)·강사별 점수·주관식 원문까지
  개별 응답 단위로 완전 복원. (당시 export 인코딩: 표시값 = 6 − 저장점수, 인구통계 = 보기순번_0)
- **헌법과 생활법률 1기**: 원시 응답 파일이 없어, 문항별/강사별/인구통계별 **분포가 통계 export 와
  정확히 일치**하도록 29건을 재구성. 집계 화면의 모든 수치(평균·분포·만족이상%)는 원본과 동일하지만,
  한 응답자 안에서의 문항 간 조합은 원본과 다를 수 있음. 주관식 의견은 원문 그대로 복원.
- 두 과정 모두 응답률 100%(설문대상=참여, hwpx 분석보고서와 일치)라 수강생은 교번 1~N(무기명,
  당시 업로드 파일과 동일)으로 완료 처리.
- 복원 문서에는 `restored: true`, `restoredAt`, `restoredSource` 마커를 넣어 식별 가능.
